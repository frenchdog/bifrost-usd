# `get_prim_asset_info`

Returns the asset identifier, name, and version.

## Inputs

### `stage`
The USD stage. 

### `path`
The path to the USD prim. 

## Outputs

### `asset_identifier`
The asset identifier. 

### `asset_name`
The name of the asset. 

### `asset_version`
The version of the asset.
