# `set_prim_asset_info`

Sets the model's info dictionary for the specified prim.

## Inputs

### `stage`
The USD stage. 

### `path`
The path to the USD prim. 

### `asset_identifier`
The asset identifier. 

### `asset_name`
The name of the asset. 

### `asset_version`
The version of the asset.
