# `prim_clear_instanceable`

Removes the authored "instanceable" opinion for this prim.

## Inputs

### `stage`
The USD stage in which to clear the instanciability. 

### `prim_path`
The path to the USD prim with metadata. 

## Outputs

### `out_stage`
The modified USD stage. 

### `success`
Boolean indicating whether the operation was successful.
