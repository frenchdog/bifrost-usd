# `clear_attribute`

Clears the authored default value and all time samples for an attribute.

## Inputs

### `stage`
The USD stage. 

### `prim_path`
The path of the prim with the attribute. 

### `name`
The name of the attribute. 

## Outputs

### `out_stage`
The modified stage. 

### `success`
Boolean indicating whether the operation was successful.
