# `set_prim_metadata`

Sets the value of a prim's metadata.

## Inputs

### `path`
The path to the USD prim. 

### `key`
The metadatum key. 

### `value`
The metadatum value. 

### `stage`
The stage in which to set the metadatum. 

## Outputs

### `out_stage`
The modified USD stage. 

### `success`
Boolean indicating whether the operation was successful.
